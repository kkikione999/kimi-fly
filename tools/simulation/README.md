# 软件在环(SIL)仿真测试框架

用于在无硬件情况下验证飞控逻辑。

## 文件说明

| 文件 | 功能 |
|------|------|
| `drone_dynamics.py` | 四旋翼动力学模型 |
| `sil_wrapper.py` | 飞控SIL封装 |
| `test_runner.py` | 自动化测试套件 |
| `README.md` | 本文件 |

## 使用方法

### 1. 运行自动化测试
```bash
python3 test_runner.py
```

### 2. 仿真模型参数

| 参数 | 数值 | 说明 |
|------|------|------|
| 质量 | 50g | 整机质量 |
| 臂长 | 6.5cm | 电机到中心距离 |
| 最大推力 | 2g/电机 | 单电机最大推力 |
| 控制频率 | 1kHz | 与真实飞控一致 |

## 预期结果

所有测试应通过，验证飞控逻辑正确性。
