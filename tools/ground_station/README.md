# Drone WiFi Ground Station

地面站测试工具，用于通过WiFi连接无人机并进行飞行控制测试。

## 功能特性

- TCP连接到ESP32-C3 WiFi模块
- 键盘控制无人机 (油门/横滚/俯仰/偏航)
- ARM/DISARM控制
- 飞行模式切换
- 实时遥测数据显示 (姿态/电机输出)
- 心跳保持连接

## 安装要求

- Python 3.7+
- 无需第三方库 (使用标准库)

## 使用方法

### 1. 启动无人机
- 确保无人机已上电
- ESP32-C3已连接到WiFi网络 (whc/12345678)
- 无人机IP地址: 192.168.4.1
- 端口: 8888

### 2. 连接到无人机网络
在电脑上连接到WiFi网络: `whc` / `12345678`

### 3. 运行地面站
```bash
cd tools/ground_station
python3 ground_station.py
```

或使用自定义IP和端口:
```bash
python3 ground_station.py --host 192.168.4.1 --port 8888
```

### 4. 键盘控制

| 按键 | 功能 |
|------|------|
| `w` / `s` | 油门 上/下 |
| `a` / `d` | 横滚 左/右 |
| `i` / `k` | 俯仰 前/后 |
| `j` / `l` | 偏航 左/右 |
| `space` | ARM/DISARM 切换 |
| `m` | 切换飞行模式 |
| `v` | 请求版本信息 |
| `t` | 请求状态信息 |
| `r` | 重置摇杆到中立位置 |
| `q` | 退出程序 |

## 显示说明

地面站会显示以下实时信息:
- RC输入值 (油门/横滚/俯仰/偏航)
- 姿态数据 (Roll/Pitch/Yaw角度)
- 电机输出 (4个电机的PWM值)
- 当前飞行模式和解锁状态

## 通信协议

使用二进制协议与无人机通信:
- 帧头: 0xAA55
- CRC16-CCITT校验
- 支持命令: ARM, DISARM, MODE, RC_INPUT
- 遥测数据: ATTITUDE, MOTOR, STATUS

详见 `firmware/stm32/comm/protocol.h`

## 安全注意事项

1. **USB供电时电机动力不足** - 无法起飞，仅用于测试
2. **飞行测试时必须使用电池供电**
3. **ARM前确保油门在最低位置**
4. **保持安全距离，小心螺旋桨**
5. **WiFi信号丢失会自动DISARM**

## 故障排除

### 无法连接
- 检查是否已连接到`whc` WiFi网络
- 确认无人机IP地址正确 (默认192.168.4.1)
- 检查ESP32-C3是否正常运行

### 无遥测数据
- 检查通信协议版本是否匹配
- 查看调试串口输出 (460800 baud)
- 确认无人机已正确初始化

### 控制无响应
- 确认无人机已ARM
- 检查是否在正确的飞行模式
- 查看是否有错误响应 (NACK)
