# Drone Flight Controller - Architecture Map

> **项目目标**: 通过WiFi控制无人机飞行
> **硬件平台**: STM32F411CEU6 + ESP8266 WiFi模块
> **开发模式**: AI自主开发，渐进式迭代

---

## 核心架构

```
┌─────────────────────────────────────────────────────────────┐
│                        WiFi 控制层                          │
│                   (ESP32-C3 - STA模式)                    │
│                   WiFi: whc / 12345678                      │
└───────────────────────────┬─────────────────────────────────┘
                            │ UART/SPI
┌───────────────────────────▼─────────────────────────────────┐
│                      飞行控制核心                           │
│                  (STM32F411CEU6)                            │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────────────┐  │
│  │   IMU驱动   │  │  气压计驱动  │  │    电机PWM控制      │  │
│  │ ICM-42688-P │  │  LPS22HBTR  │  │   (空心杯电机×4)    │  │
│  │   (SPI3)    │  │   (I2C1)    │  │                     │  │
│  └─────────────┘  └─────────────┘  └─────────────────────┘  │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────────────┐  │
│  │  磁力计驱动  │  │   AHRS算法  │  │     PID控制器       │  │
│  │  QMC5883P   │  │  姿态解算   │  │  (角度/角速度环)    │  │
│  │   (I2C1)    │  │             │  │                     │  │
│  └─────────────┘  └─────────────┘  └─────────────────────┘  │
└─────────────────────────────────────────────────────────────┘
```

## 开发阶段

1. **Phase 1**: HAL层基础 (GPIO/PWM/UART/I2C/SPI)
2. **Phase 2**: 传感器驱动 (IMU/气压计/磁力计)
3. **Phase 3**: 姿态解算 (AHRS算法)
4. **Phase 4**: 飞控算法 (PID控制器)
5. **Phase 5**: 通信协议 (WiFi/命令协议)
6. **Phase 6**: 整合测试与飞行调试
