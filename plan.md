# kimi-fly WiFi 无人机开发计划

## 目标概述
开发一款可通过 WiFi 控制的四轴无人机，使用 STM32+ESP32-C3 双处理器架构，实现稳定的飞行控制和无线通信。

## 硬件架构
- **主处理器**: STM32 (F411/F405/H743) - 飞行控制算法、传感器融合、电机控制
- **协处理器**: ESP32-C3 - WiFi通信、遥控指令转发、数据记录
- **传感器组**: IMU (MPU6050/ICM42688)、气压计(BMP280)、磁力计(QMC5883)、GPS(NEO-M8N)
- **动力系统**: 4x 无刷电机 + BLHeli_S/AM32 电调 (DShot协议)

## 代码规范
- 遵循 CLAUDE.md 中的 C11 编码标准
- 禁止飞行代码中使用动态内存分配
- 中断服务程序必须简短 (<10μs)
- 所有硬件访问必须通过 HAL 层
