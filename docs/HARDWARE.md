# Hardware Architecture

kimi-fly 无人机硬件架构文档。

## 系统概览

```
┌─────────────────────────────────────────────────────────────┐
│                        kimi-fly                             │
├─────────────────────────────────────────────────────────────┤
│  机身 (Airframe)          │  主控 (Flight Controller)        │
│  ├── 机架结构              │  ├── STM32 (主处理器)            │
│  ├── 动力系统              │  │   ├── 飞行控制算法            │
│  │   ├── 电机 ×4          │  │   ├── 传感器融合              │
│  │   ├── 电调 ×4          │  │   └── 电机控制                │
│  │   └── 电池             │  ├── ESP32-C3 (协处理器)         │
│  ├── 螺旋桨               │  │   ├── 无线通信 (WiFi/BLE)     │
│  └── 起落架               │  │   ├── 遥控接收                │
│                           │  │   └── 数据记录                │
│                           │  └── 传感器组                     │
│                           │      ├── IMU (MPU6050/ICM42688) │
│                           │      ├── 气压计 (BMP280)        │
│                           │      ├── 磁力计 (QMC5883)       │
│                           │      └── GPS (NEO-M8N)          │
└─────────────────────────────────────────────────────────────┘
```

## 主控 (Flight Controller)

### 目录
- `主控/`: 原理图、PCB 设计文件

### 处理器

#### STM32 (主处理器)

待确定具体型号：
- 候选: STM32F411, STM32F405, STM32H743
- 要求: FPU, 足够 GPIO, 多路 UART/SPI/I2C

#### ESP32-C3 (协处理器)

- 型号: ESP32-C3-WROOM-02
- 功能: 无线通信、数据转发

### 传感器

| 传感器 | 接口 | 用途 | 状态 |
|--------|------|------|------|
| IMU | SPI/I2C | 姿态感知 | 待选型 |
| 气压计 | I2C | 高度估计 | 待选型 |
| 磁力计 | I2C | 航向参考 | 待选型 |
| GPS | UART | 位置定位 | 待选型 |

### 通信接口

| 接口 | 用途 |
|------|------|
| UART1 | 调试/CLI |
| UART2 | GPS |
| UART3 | ESP32-C3 |
| SPI1 | IMU (高速) |
| I2C1 | 气压计、磁力计 |
| CAN | 预留 (未来扩展) |
| USB | 固件更新、调试 |

## 电源架构

```
电池 (LiPo 4S, 14.8V)
    │
    ├──→ 5V BEC ──→ 主控板 (STM32, ESP32, 传感器)
    │
    └──→ 电调 ──→ 电机
```
