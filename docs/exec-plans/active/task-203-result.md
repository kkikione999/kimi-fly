## 技术调试记录

### 已解决问题

1. **HardFault (undefined instruction)** - 链接脚本缺少 `.init_array`/`.fini_array` 节，导致 `__libc_init_array` 跳转到无效地址
   - 修复：在 `build.sh` 链接脚本中添加所有 init/fini array 节

2. **HardFault (PLL 初始化)** - 代码使用 HSE (外部晶振) 但硬件无晶振，PLL 超时崩溃
   - 修复：改为 HSI (16MHz 内部振荡器) + PLL = 100MHz SYSCLK

3. **浮点数输出空白** - `nano.specs` 默认不包含浮点 printf 支持
   - 修复：添加 `-u _printf_float` 链接标志

4. **UART 无输出** - ST-Link VCP 对应 USART1 (PA9/PA10)，程序启动后立即输出被错过
   - 修复：添加 `HAL_Delay(2000)` 在首次 UART 输出前

5. **IMU I2C 地址错误** - 文档标注 0x68，实际硬件 AD0 为高电平，地址为 0x69
   - 修复：将 `ICM42688_ADDR` 改为 0x69

### 待解决问题

| 问题 | 影响 | 优先级 |
|------|------|--------|
| 气压计压力读数偏低约 3.5 倍 | 气压计不可用 | 高 |
| 磁力计 Y/Z 轴恒为 0 | 磁力计方向感知受损 | 高 |
| LPS22HH 与 LPS22HB 驱动差异 | 需专用驱动或配置校正 | 高 |
