## 项目概览

本项目实现了**完整的无人机WiFi飞行控制系统**，从底层硬件抽象到上层飞行控制，再到地面站工具，全部自主设计实现。

### 核心成就

- **27个源代码文件**，完整实现飞控系统
- **1kHz控制频率**的实时飞行控制
- **WiFi远程控制**，支持手机/PC地面站
- **完整级联PID** (角度环+角速度环)
- **9轴姿态解算** (加速度计+陀螺仪+磁力计)

## 交付物清单

### 1. 硬件抽象层 (HAL) - 5个模块

| `hal/i2c.c/h` | I2C总线 (100kHz) | ⭐⭐⭐ |
| `hal/spi.c/h` | SPI通信 (SPI3) | ⭐⭐⭐ |

**总计**: ~2000行代码

### 2. 传感器驱动 - 3个驱动

| 文件 | 传感器 | 接口 | 功能 |
|------|--------|------|------|
| `drivers/icm42688.c/h` | ICM-42688-P (IMU) | I2C1 @ 0x68 | 6轴惯性测量 |
| `drivers/lps22hb.c/h` | LPS22HBTR (气压计) | SPI3 | 气压/温度 |
| `drivers/qmc5883p.c/h` | QMC5883P (磁力计) | I2C1 @ 0x2C | 3轴磁场 |

**总计**: ~1500行代码

### 3. 姿态解算算法 (AHRS) - 4个模块

| 文件 | 功能 | 算法 |
|------|------|------|
| `algorithm/sensor_calibration.c/h` | 传感器校准 | 6面校准、零偏校准 |

| `algorithm/pid_controller.c/h` | PID控制器 | 位置式/增量式+抗饱和 |
| `algorithm/flight_controller.c/h` | 飞行控制 | 级联PID+X型混控 |

**总计**: ~2500行代码

## 关键技术参数

### 控制参数
- **控制频率**: 1kHz (1ms周期)
- **AHRS更新**: 1kHz (与主循环同步)
- **WiFi处理**: 200Hz (5ms)
- **遥测发送**: 50Hz (20ms)
- **状态报告**: 1Hz

### PID配置 (默认值)
| 通道 | Kp | Ki | Kd | 输出限制 |
|------|-----|-----|-----|----------|
| ROLL_ANGLE | 4.0 | 0.05 | 0.0 | ±200°/s |
| ROLL_RATE | 0.15 | 0.3 | 0.001 | ±400 |
| PITCH_ANGLE | 4.0 | 0.05 | 0.0 | ±200°/s |
| PITCH_RATE | 0.15 | 0.3 | 0.001 | ±400 |
| YAW_ANGLE | 3.0 | 0.02 | 0.0 | ±150°/s |
| YAW_RATE | 0.2 | 0.4 | 0.0 | ±300 |

### 通信协议
- **帧头**: 0xAA55
- **校验**: CRC16-CCITT
- **最大负载**: 64字节
- **波特率**: 115200 (STM32-ESP32)

## 使用指南

### 1. 编译固件

```bash
cd firmware/stm32
# 使用STM32CubeIDE或Makefile编译
make all
```

### 2. 烧录固件

```bash
# 通过ST-Link烧录
st-flash write build/flight_controller.bin 0x8000000
```

### 3. 连接地面站

```bash
# 1. 连接WiFi: whc / 12345678
# 2. 运行地面站
cd tools/ground_station
python3 ground_station.py

# 3. 使用键盘控制
# w/s - 油门, a/d - 横滚, i/k - 俯仰, j/l - 偏航
# space - ARM/DISARM, m - 切换模式
```

## 注意事项

### 安全警告
1. **USB供电时无法起飞** - 电机动力不足
2. **飞行测试必须使用电池**
3. **ARM前确保油门最低**
4. **保持安全距离**

### 已知限制
- PID参数为经验值，需要实际飞行调试
- 室内磁力计可能有干扰
- WiFi延迟约10-50ms
